# HeroesApp

SPA desarrollada en Angular para la gestión de héroes, con autenticación de usuarios y backend simulado usando JSON-Server.

## Tecnologías utilizadas

- Angular 16
- Node.js v18
- Angular Material
- PrimeNG
- PrimeFlex
- JSON-Server

## Instrucciones de instalación

1. Clonar el repositorio:
    ```
    git clone https://github.com/Omar2904-lgtm/heroesApp.git
    ```
2. Instalar dependencias:
    ```
    npm install
    ```
3. Levantar backend simulado:
    ```
    npm run backend
    ```
   Esto corre JSON-Server en `http://localhost:3000` usando `data/db.json`.

4. Ejecutar la aplicación Angular:
    ```
    ng serve -o
    ```
   Accede a la app en `http://localhost:4200`.
